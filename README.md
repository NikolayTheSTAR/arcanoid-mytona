# arkanoid-mytona
 
Тестовое задание для MyTona.

Реализованные сущности:

 * платформа, управляемая игроком
 * летающий шарик
 * блоки
 * бонусы (4 шт.)

 Дополнительно:

 * генерация цветов блоков
 * не ограниченное количество уровней
 * очки и запись рекорда
 * сборка под Android
 * XML-документация

Время реализации - 3 дня.
